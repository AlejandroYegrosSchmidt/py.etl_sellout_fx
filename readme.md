# ETL Sell Out FX 📊

Proyecto automatizado para la carga de datos de Sell Out desde archivos Excel a una base de datos PostgreSQL.

## 🚀 Tecnologías
* **Python** (Pandas, SQLAlchemy)
* **Streamlit** (UI)
* **Docker & Docker Compose** (Contenedorización)
* **PostgreSQL** (Base de Datos)

## 🛠️ Cómo ejecutar
1. Clona el repositorio.
2. Asegúrate de tener Docker abierto.
3. Ejecuta: `docker-compose up --build`
4. Abre `http://localhost:8501` en tu navegador.
